# Post para LinkedIn

## Versão sugerida para o post

Estou publicando um projeto open source que criei para ajudar profissionais que querem melhorar o perfil no LinkedIn, mas não têm como pagar por uma avaliação especializada.

A ideia nasceu de um ponto simples: muita gente boa fica invisível para recrutadores por falta de posicionamento, palavras-chave, clareza de impacto e organização do perfil.

Esse plugin foi feito de desenvolvedor para desenvolvedor, com foco inicial em tecnologia, para entregar uma análise prática do perfil com:

- score de mercado
- senioridade estimada
- pontos fortes
- pontos fracos
- problemas identificados
- sugestões prioritárias
- exportação em PDF

Em um dos testes recentes, o diagnóstico retornou `71%` de aderência, com senioridade estimada em `Pleno`, mostrando de forma objetiva onde o perfil já comunica valor e onde ainda precisa evoluir.

O objetivo do projeto é democratizar esse tipo de análise para a comunidade, especialmente para quem está em transição, procurando recolocação ou simplesmente não consegue investir em consultorias e avaliações pagas.

## Stack utilizada

- React
- TypeScript
- Vite
- Chrome Extension
- Node.js
- Express
- Groq API para enriquecimento da análise com IA
- Vitest
- React Testing Library
- jsPDF
